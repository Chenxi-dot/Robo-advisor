# 智能投资研究平台 (Intelligent Investment Research Platform)

## 📖 项目简介
本项目是一个基于 Python 和 Streamlit 开发的综合性 A 股投资研究工具。它集成了实时行情数据、深度财务分析、舆情监控以及基于大语言模型 (LLM) 的智能投顾功能。旨在为投资者提供一站式的数据查询与辅助决策支持。

## ✨ 核心功能

### 1. 🌏 市场全景
- **实时指数**: 展示上证指数、深证成指、创业板指等核心指数的实时涨跌情况。
- **市场热点**: (示例) 展示当前市场的热门行业与资金流向。

### 2. 🔍 个股深度研究
- **K线走势**: 提供交互式的日 K 线图，包含均线 (MA5, MA20) 和成交量分析。
- **基本面分析**: 展示公司总市值、流通市值、市盈率 (TTM)、市净率等核心估值指标。
- **财务报表**: 
    - 可视化的关键财务指标趋势 (营收、净利润)。
    - 完整的利润表、资产负债表、现金流量表摘要。
- **舆情新闻**: 
    - 实时抓取东方财富股吧热帖。
    - 公司最新公告与机构研报。
- **行业对比**: 
    - 自动识别所属行业，获取同行业成分股。
    - 提供市值排名、净利润排名、涨跌幅排名。
    - 行业全景图 (Treemap) 与 估值分布图 (Box Plot)。
- **🤖 AI 智能投顾**: 
    - **多智能体系统**: 包含基本面分析师、技术分析师、新闻分析师、风险控制专家。
    - **深度分析**: 基于 Qwen-2.5-7B 模型，结合实时数据生成专业的投资分析报告。
    - **对话功能**: 支持与 AI 投顾团队进行多轮对话，解答个性化投资疑问。

### 3. 💼 投资组合助手
- **仓位计算**: 根据总预算与当前股价，自动计算最大可买手数与资金占用。
- **风险对冲**: 基于当前市场环境，利用 AI 生成结构化的风险对冲策略建议。

## 🛠️ 技术栈
- **前端框架**: Streamlit
- **数据源**: AkShare (主要对接东方财富、新浪财经接口)
- **数据处理**: Pandas, NumPy
- **可视化**: Plotly Interactive Charts
- **大模型**: Qwen-2.5-7B (通过 OpenAI 兼容接口调用)

## 🚀 快速开始

### 环境要求
- Python 3.8 或更高版本
- 建议使用 Anaconda 或 Virtualenv 管理环境

### 1. 安装依赖
在项目根目录下运行：
```bash
pip install -r requirements.txt
```

### 2. 配置 API Key
本项目使用 Qwen 模型，请确保在 `llm_utils.py` 中配置了正确的 API Key 和 Base URL，或者设置环境变量。

### 3. 运行应用
```bash
streamlit run investment_research.py
```
应用启动后，浏览器将自动打开 `http://localhost:8501`。

## 📂 文件结构
- `investment_research.py`: 主程序入口。
- `agents.py`: 定义 AI 分析师角色的类。
- `llm_utils.py`: 处理 LLM API 调用。
- `stock_list.csv`: 本地缓存的股票列表文件。
- `TECHNICAL_DOCS.md`: 技术实现详细文档。
